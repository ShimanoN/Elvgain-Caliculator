## モチベーション維持のコツ

1. **視覚化**: 週目標ページでチャートを毎日確認
2. **ルーティン化**: トレーニング後すぐに入力
3. **報酬設定**: 週目標達成ごとに自分にご褒美
4. **進捗共有**: GitHubやSNSで進捗を共有
5. **柔軟性**: 休息日は0mでも入力してストリークを保つ

## 緊急時の対応

### データが消えた場合
1. 最新のバックアップを確認
2. Firebase Consoleでデータを確認
3. ブラウザのIndexedDBを確認
4. GitHub Issuesに報告

### アプリが動かない場合
1. ブラウザのキャッシュをクリア
2. 別のブラウザで試す
3. ローカル環境で起動（`npm run dev`）
4. エラーログを取得してGitHub Issuesに報告
