## 付録A: 用語解説（IT初心者向け）

- **MVP（Minimum Viable Product）**: 最小限の機能を持つ製品。ユーザーに価値を提供できる最小単位。
- **CI/CD**: 継続的インテグレーション/継続的デプロイ。コード変更を自動的にテスト・デプロイする仕組み。
- **テストカバレッジ**: テストによって検証されているコードの割合。80%以上が優秀。
- **デプロイ**: アプリケーションを本番環境に配置し、ユーザーが使える状態にすること。
- **IndexedDB**: ブラウザ内のデータベース。オフラインでもデータを保存できる。
- **CodeQL**: GitHubが提供するセキュリティ分析ツール。脆弱性を自動検出。
